# Dream Visualization

一个展示睡眠阶段和梦境的可视化网站。

## 功能特点

- 交互式睡眠阶段展示
- 动态背景效果
- 视频播放功能
- 响应式设计

## 技术栈

- Next.js
- React
- TypeScript
- CSS Animations

## 本地开发

1. 克隆仓库
```bash
git clone [你的仓库地址]
```

2. 安装依赖
```bash
npm install
# 或
yarn install
```

3. 运行开发服务器
```bash
npm run dev
# 或
yarn dev
```

4. 打开 [http://localhost:3000](http://localhost:3000) 查看网站

## 部署

本项目使用 Vercel 进行部署。每次推送到主分支时都会自动部署。
